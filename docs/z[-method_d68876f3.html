[,Model,character,missing,missing-method overload the [ ] operator for models and character vector

Description

This method overloads the [] operator for Model objects and acts as an alternative to the get methods so m["times"] is equivalent to getTimes(m), m["C"] to getC(m) and so on.

Usage

## S4 method for signature 'Model,character,missing,missing'
x[
i]

Arguments

x

: of class Model

i

: of class character
