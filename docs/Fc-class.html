Fc S4 class

Description

The 14C fraction is a necessary ingredient of any Model_14 object. In the most general case it is a real valued function of time, accompanied by a string describing the unit or format (i.e. "Delta14C" or "afn" for Absolute Fraction Modern) . In the most simple case it is constant real number plus format.
